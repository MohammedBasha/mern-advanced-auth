<h1 align="center">MERN Advanced Auth 🔐</h1>

<p align="center">
  A full-stack authentication system built with the MERN stack featuring secure JWT authentication, email verification, password reset functionality, and protected routes.
</p>

---

# 🚀 Features

- 🔐 Authentication using JWT
- 📧 Email Verification with 6-Digit OTP Code
- 🔑 Forgot Password & Reset Password Functionality
- 🍪 Secure HTTP-Only Cookie Authentication
- 🔒 Password Hashing with bcryptjs
- 🛡️ Protected Routes & Middleware
- 📩 Transactional Emails with Mailtrap
- ⚡ Frontend built with React + Vite
- 🎨 Responsive UI using Tailwind CSS
- 🗄️ MongoDB Database Integration
- 🌐 REST API with Express.js
- 🔄 Persistent Authentication State
- 🚪 Secure Logout System
- ✅ Form Validation & Error Handling

---

# 🛠️ Tech Stack

## Frontend

- React
- Vite
- Tailwind CSS

## Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (jsonwebtoken)
- bcryptjs
